Introduction to ADBShell

ADBShell is a wrapper for ADB & Fastboot, instead of learning all commands and keeping "adb help" and "fastboot help" at all times, use ADBShell!

ADBShell makes commands like: "adb shell getprop" to: "check"

Instead of: adb reboot fastboot, do: "reboot fastboot" (Could'nt find a difference :))

Why you should use it

It makes me happy

It simplifies using the ADB toolchain, instead of knowing all commands, it just needs you to type a command and it executes the command!
